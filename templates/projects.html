My Website. Made with Flask

This is my personal website which contains a description about me and what I like to do. I created this using Flask and pulling from bootstrap examples and making them my own.

Simple HTTP Server Functionality: GET,PUT,HEAD

This is a simple http server that I created that is able to recieve http requests, parse the request and complete each request, GET: sends client the requested file if it is currently on the server. If not returns a http error for client to parse. PUT: client sends a file to my server and I store the file onto my server. HEAD: sends the client back the size of the file data it called the function on. If it is none of these requests then I would send a http bad request error back to the client.

Multi-threaded HTTP server Checkmate.
